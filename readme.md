# Vite React Tailwind Template

This template provide toolchain below:

- Vite
- React, ReactDOM
- ESlint, Prettier
- Git Hook (simple-git-hook, Lint Staged)
- TailwindCSS 4
- daisyui
- React Router DOM (auto generated routes)

# Usage

```sh
pnpm i
pnpm dev
```
